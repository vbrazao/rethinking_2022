We will run the following model with quap:

\[ \text{weight}_i \sim \operatorname{Normal}(\mu_i, \sigma) \] \[ \mu_i = \alpha + \beta \text{height}_i\] \[ \alpha \sim \operatorname{Normal}(0, 1) \] \[ \beta \sim \operatorname{Log-Normal}(0, 1) \] \[ \sigma \sim \operatorname{Exponential}(\frac{1}{2}) \] Letâs do some prior predictive simulation to check that this makes some sense.

one_prior_lines <- tibble::tibble(
  n = 1:1e2,
  intercept = rnorm(n = 1e2, mean = 0, sd = 1),
  slope = rlnorm(n = 1e2, mean = 0, sd = 1)
) %>% 
  tidyr::expand(tidyr::nesting(n, intercept, slope), height = range(d1_z$height)) %>% 
  dplyr::mutate(
    weight = intercept + slope * height
  )

one_prior_lines %>% 
  ggplot2::ggplot(ggplot2::aes(x = height, y = weight, group = n)) +
  ggplot2::geom_line(alpha = .2)

This captures the idea that negative relationships are not possible while still allowing for some absurdly large slopes. Letâs also simulate some data.

For the total causal effect, we simply need to regress weight on age.

Two things are important for the priors: 1. The intercept should be positive and somewhere around average weight at birth. 2. The slope should be positive.

I think we can use the Log-Normal trick here again.

For the intercept: English Wikipedia tells us that normal birthweight ranges from 2.5 to 4.5kg. The mean of that is 3.5, so that should be the mean of our prior for the intercept. Tinkering with this online calculator tells me that \(\operatorname{Log-Normal}(0.75, 1)\) has a mean of 3.49 and a standard deviation of 4.575. Iâll stick with that unless simulation shows it to be absurd.

For the slope, Iâll try the same prior as before, \(\operatorname{Log-Norma}(0, 1)\).

And so, our model:

\[ \text{weight}_i \sim \operatorname{Normal}(\mu_i, \sigma) \] \[ \mu_i = \alpha + \beta \text{age}_i\] \[ \alpha \sim \operatorname{Log-Normal}(0.75, 1) \] \[ \beta \sim \operatorname{Log-Normal}(0, 1) \] \[ \sigma \sim \operatorname{Exponential}(\frac{1}{2}) \]

Letâs again simulate many lines to see if this makes sense. Adapting the code from the previous slide to reflect the new priors and swapping height for age gives:

Ok, this seems a little excessiveâ¦ Letâs try to tighten the prior on the intercept and slope a little bit. \(\operatorname{Log-Normal}(1.1, 0.5)\) has a mean of 3.40 and a standard deviation of 1.81, which seems a bit more reasonable. For the slope, the mean should be around 2: these WHO charts (for girls and boys) make me think that is a reasonable number. \(\operatorname{Log-Normal}(0.5, 0.7)\) has a mean of 2.11 and a standard deviation of 1.68, so letâs try that.

The new model:

\[ \text{weight}_i \sim \operatorname{Normal}(\mu_i, \sigma) \] \[ \mu_i = \alpha + \beta \text{age}_i\] \[ \alpha \sim \operatorname{Log-Normal}(1.1, 0.5) \] \[ \beta \sim \operatorname{Log-Normal}(0.5, 0.7) \] \[ \sigma \sim \operatorname{Exponential}(\frac{1}{2}) \]
